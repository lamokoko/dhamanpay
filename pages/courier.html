GPS Slide

How GPS Insertion Works

Step by Step

Insert Position

1

When the courier opens the shipping form, the page can call navigator.geolocation.getCurrentPosition() to read the device latitude and longitude.

2

These values can fill the visual pickup and live-tracking fields in the current interface.

3

The current documented backend does not expose a GPS tracking endpoint, so this section works as UI guidance and local preview only.

4

If you add GPS endpoints later, you can connect this same layout to live tracking without redesigning the dashboard.
